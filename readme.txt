== Description ==

Provides a cost calculator for undergraduate students.

== Installation ==

1. Upload the files to the `/wp-content/plugins/undergrad-calculator/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the [cost-calculator] short code in any page or post
